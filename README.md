# deixe-de-ser-noob-em-git
Repositório pra você aprender git

## Tutorial

1. Clonar repositório

`git clone git@github.com:araujo88/deixe-de-ser-noob-em-git.git`

2. Criar novo branch

`git checkout -b <nome-do-branch>`

3. Editar arquivo Contribuidores.md

Abra o arquivo Contribuidores.md usando seu editor favorito e adicione seu nome abaixo dos demais.

4. Adicione o arquivo pra staging

`git add Contribuidores.md`

5. Faça o commit

`git commit -m "Uma mensagem informativa do que vc fez"`

6. Faça o pull request

`git push origin <nome-do-branch>`
